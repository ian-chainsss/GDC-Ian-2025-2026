# Encryptie Onderzoek
Deze folder bevat een onderzoek en overzicht van verschillende encryptie-algoritmes, hun werking, toepassingen en beveiligingsaspecten. Het doel is om een gestructureerd inzicht te bieden in de wereld van encryptie, zodat ontwikkelaars en beveiligingsprofessionals beter geïnformeerde keuzes kunnen maken bij het implementeren van beveiligingsmaatregelen.  

De folder bevat de volgende onderdelen:
- `README.md`: Dit bestand, dat je nu leest, biedt een overzicht van de inhoud van deze folder en de structuur van het onderzoek.
- `lijst.md`: Een lijst van alle encryptie-algoritmes die in dit onderzoek worden behandeld. Daarbij ook een lijst van de meest belangrijkste die verder onderzocht worden in de andere bestanden.
- `overzicht.md`: Een overzicht van alle encryptie-algoritmes, ingedeeld in hoofdcategorieën en subcategorieën op basis van hun werkingsmechanisme en gebruik.
- `werking.md`: Een diepgaande uitleg van de werking van de belangrijkste encryptie-algoritmes, inclusief wiskundige principes en voorbeelden.
- `JWTs.md`: Een specifiek onderzoek naar handtekeningen, signing en encryptie bij JSON Web Tokens (JWTs), inclusief best practices en beveiligingsaanbevelingen.
